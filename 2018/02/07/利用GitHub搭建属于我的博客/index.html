利用GitHub搭建属于我的博客

2018-02-07
  • blog

寒假实在太无聊，于是逛各种网站偶然翻到一个搭建博客的网页，非常羡慕，于是我也动手去尝试了。文笔不好，大佬轻喷……

准备行动

利用hexo搭建博客,那么hexo是什么呢？我不做过多详细介绍了，大家可以去阅读官方文档
需要以下框架

node.js

git

关于node.js和git的相关介绍在他们的官网也有，我就不说废话了。

开始搭建

tips：关于gitbash的默认位置建议大家修改下不用每次都要重新用cd命令选择路径
修改方法：
安装好git后打开gitbash

输入命令

$ npm install -g hexo-cli

回车等待就安装好hexo框架了

紧接着就输入以下命令初始化hexo

$ hexo init <文件夹>
$ cd <文件夹>

$ npm install
