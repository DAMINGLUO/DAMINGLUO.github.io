    博客搭建记（一）

    搭建博客很久了，一直没有写自己的博客
    今天终于有时间写博客了，准备先写下博客搭建过程
    我使用的是hexo框架搭建的博客
    在搭建博客之前需要先下载
    1.git
    2.node.js
    下载好后你可以选择一个比较空余的的路径安装好

    安装好git后打开gitbash

    输入命令

    $ npm install -g hexo-cli

    回车等待就安装好hexo框架了

    紧接着就输入以下命令初始化hexo

    $ hexo init <文件夹>
    $ cd <文件夹>

    $ npm install

    以及建议记住以下命令
    — bash
    cd 路径 打开相应路径
    git init “文件夹名字” 新建一个文件夹

    git clone “仓库名” 把仓库克隆下来

    注：双引号不用输入

    建议学会git命令可以通过以下方式学习
